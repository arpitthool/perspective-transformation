# Perspective Transformation

A C++ application for applying top-view perspectice transformation to a selected region in a given video using OpenCV.

## Requirements

- OpenCV 4.x
- CMake 3.10+
- C++11 compatible compiler (g++, clang++, etc.)

## Building

```bash
mkdir build
cd build
cmake ..
make
```

## Running

```bash
cd build
./main
```
